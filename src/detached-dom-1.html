Detached DOM patterns

pattern 1 (not leak)

  • use `document.createElement`
  • and assign one variable

      var detachedDom;
      document.getElementById('b1').addEventListener('click', function(){
        detachedDom = document.createElement('div')
        console.count('created Detached DOM')
      })

pattern 2 (leak)

  • use `document.createElement`
  • and assign array

      var detachedDoms = []
      document.getElementById('b2').addEventListener('click', function(){
        detachedDoms.push(document.createElement('div'))
        console.count('created Detached DOM')
      })
